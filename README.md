# Sign Language Recognition using EfficientNet-B0

## Overview
This project is a deep learning-based Sign Language Recognition system developed using PyTorch and the EfficientNet-B0 model. It classifies American Sign Language (ASL) alphabet gestures from images.

## Features
- Image classification using EfficientNet-B0
- PyTorch implementation
- ASL alphabet recognition
- Image preprocessing and prediction
- Model training and evaluation

## Technologies Used
- Python
- PyTorch
- Torchvision
- EfficientNet-PyTorch
- NumPy
- Matplotlib
- Pillow

## Dataset
This project uses the **ASL Alphabet Dataset**.
